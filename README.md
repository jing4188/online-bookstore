# 书店管理系统

一个基于 Flask 的书店管理系统，包含用户端和管理员端。

## 项目结构

- `bookstore_user/` - 用户端应用
- `bookstore_admin/` - 管理员端应用
- `templates/` - 共享模板文件
- `static/` - 静态资源文件

## 功能特性

### 用户端功能
- 用户注册和登录
- 浏览所有书籍
- 查看书籍详情
- 购物车管理
- 下单和支付
- 订单管理
- 评价系统
- 个人资料管理

### 管理员端功能
- 管理员登录
- 用户管理
- 书籍管理（添加、编辑、删除）
- 订单管理
- 评论管理
- 反馈管理

## 运行应用

### 方法一：分别启动

1. 启动用户端应用：
   ```bash
   cd bookstore_user
   python app.py
   ```
   访问地址：http://192.168.3.193:3000/

2. 启动管理员端应用：
   ```bash
   cd bookstore_admin
   python app.py
   ```
   访问地址：http://192.168.3.193:5001/

### 方法二：使用启动脚本

运行以下命令同时启动两个应用：
```bash
python start_all.py
```

## 默认账号

- 管理员账号：`admin`
- 管理员密码：`admin123`

## 数据库

项目使用模拟数据库（JSON文件存储），无需安装 MongoDB 即可运行。

- 管理员端：5001
